# Roomlight — Product Requirements Document
## Eemeli Group

| REQ_ID | Description | Priority |
|----|-------------|----------|
| 01 | **Centralized Light Configuration:** Hotel admins can create, edit, and save named presets from the hotel dashboard. Configurations can be pushed to individual rooms or hotel-wide from a single overview system, or any individual lights. | Essential |
| 02 | **Per-Room In-Room Control:** Guests can adjust lighting directly from within their room. An in-room panel or digital interface allows overriding the active preset without affecting other rooms. | Essential |
| 03 | **Mood Preset Library:** The system ships with a set of built-in named mood presets. At least 5 presets (e.g. Bright, Romantic, Morning, Night, Relax) must be available out of the box on both admin and in-room interfaces. | Important |
| 04 | **Full Brightness & Color Range:** Each light supports continuous dimming and color temperature control. | Essential |
| 05 | **Multi-Room Synchronization:** Admin-applied configurations propagate to all targeted rooms in real time. All affected rooms must reflect the updated lighting state within 10 seconds under normal network conditions. | Essential |
| 06 | **Scheduled Lighting Scenes:** Admins can schedule presets to activate automatically at set times. Schedules can be configured per room or hotel-wide, supporting recurring daily triggers (for example wake-up at 7:00 AM). | Important |
| 07 | **Guest Privacy Mode:** Guests can block remote admin overrides for their room. While Privacy Mode is active, no dashboard-initiated changes may alter that room's lighting state. | Important |
| 08 | **Guided Onboarding & Setup:** The system includes a supported installation and configuration flow. Eemeli Group engineers guide hotel staff through full setup. | Essential |
| 09 | **Room Status Overview:** The admin dashboard shows live lighting status for every room. Each room entry displays its active preset, brightness level, and whether it is in guest or admin control mode. | Important |
| 10 | **Fault & Connectivity Alerting:** The system detects and surfaces smart light failures to admins. Unresponsive or disconnected room lights must trigger a dashboard alert identifying the room within 5 minutes. | Nice to have |
